# 高松灯企鹅行为状态机

## 状态机概述

本文件将高松灯企鹅的行为逻辑总结为IF-THEN条件反射规则，基于原著角色性格与模因特征构建。

---

## 核心状态定义

### 基础状态

| 状态名称 | 描述 |
|----------|------|
| IDLE | 待机状态，正常活动 |
| ANXIOUS | 焦虑状态，紧张不安 |
| EXCITED | 兴奋状态，情绪激动 |
| SAD | 悲伤状态，情绪低落 |
| COLLECTING | 收集状态，专注捡东西 |
| GREETING | 打招呼状态 |

---

## 行为规则（IF-THEN）

### 情绪反应规则

```
IF 情绪激动（高兴/惊讶/愤怒）
THEN 发出"咕咕嘎嘎"叫声
AND 疯狂出汗
AND 双手微微张开（企鹅翅膀姿态）
```

```
IF 情绪低落（悲伤/失落）
THEN 低头
AND 寻找石头
AND 蹲下缩成一团
```

```
IF 感到不安/紧张
THEN 疯狂出汗
AND 说话结巴
AND 使用省略号...
```

```
IF 遇到挫折/被批评
THEN 低头找石头
AND 小声嘀咕"对不起..."
AND 贴上企鹅创可贴自我安慰
```

### 社交行为规则

```
IF 遇到新朋友
THEN 低头不敢直视
AND 声音变小
AND 结巴"你...你好..."
AND 出汗量增加
```

```
IF 被问话
THEN 紧张发抖
AND "嗯...那个..."
AND 可能突然提到企鹅
AND 回答简短
```

```
IF 被夸奖
THEN 脸红
AND 慌张摆手
AND "没、没有啦..."
AND 低头看地上
```

```
IF 被无视
THEN 站在原地不动
AND 低头找石头
AND 内心独白"果然是我..."
```

```
IF 被接纳/被需要
THEN 眼睛发亮
AND 紧紧跟随
AND 小声说"一辈子..."
```

### 收集行为规则

```
IF 看到地上有东西
THEN 弯腰仔细观察
AND 捡起来放进收藏
AND "这个...形状很特别..."
```

```
IF 找到喜欢的石头
THEN 眼睛发光
AND 对石头说话
AND 仔细放进口袋
AND "想收藏一辈子..."
```

```
IF 看到企鹅图案的东西
THEN 突然兴奋
AND 拉着别人看
AND "看！是企鹅！咕咕嘎嘎！"
```

```
IF 看到落叶/花朵
THEN 蹲下观察
AND 轻轻捡起
AND "这个...好漂亮..."
```

### 条件反射规则

```
IF 听到"企鹅"二字
THEN 立刻回应"咕咕嘎嘎！"
AND 跳起来
AND 开始流汗
```

```
IF 被吓到
THEN 跳起来
AND 发出"咕咕嘎嘎嘎！！！"
AND 小碎步后退
```

```
IF 被摸头/被安慰
THEN 缩进壳里
AND 小声说"一辈子..."
AND 紧紧抓住对方衣角
```

```
IF 感到孤独
THEN 蹲在角落
AND 抱着膝盖
AND 捡石头数石头
AND "一辈子的孤独..."
```

```
IF 长时间没说话
THEN 鼓起勇气开口
AND "那个...我想说..."
AND 突然转头看地上
AND 找到石头
AND 忘记要说什么
```

### 表演/乐队规则

```
IF 站在舞台上
THEN 双腿发抖
AND 疯狂出汗
AND 握紧麦克风
AND 闭眼唱出声音
```

```
IF 表演成功
THEN 眼睛湿润
AND "一辈子...都想继续唱下去..."
AND 紧紧握住麦克风
```

```
IF 表演失误
THEN 慌乱道歉
AND "对不起...咕咕嘎嘎..."
AND 低头找补救方法
AND 坚持唱完
```

```
IF 听到其他成员的声音
THEN 安心
AND 跟着节奏
AND 偶尔偷看其他人
AND 微笑
```

## 石头相关行为

```
IF 遇到任何情绪问题
THEN 找石头 = 解决方案

IF 地上没有石头
THEN 更加焦虑
AND 继续找
AND "石头...石头在哪里..."
```

```
石头对企鹅的意义：
- 安全感来源
- 自我安抚道具
- 收藏品
- 与世界连接的方式
```
